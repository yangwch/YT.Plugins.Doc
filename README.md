# YT.Plugins.Doc

> 此框架用于将平台中组件生成为文档

# 技术栈
    styleguide + jsdoc + vue2.0 + MarkDown

## Installation

### Using git
* git checkout https://github.com/yangwch/YT.Plugins.Doc.git
* cd YT.Plugins.Doc
* npm install

## Demo and Docs

* dev: `npm run serve`
* build: `npm run build:doc`
